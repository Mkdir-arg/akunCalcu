# REQ-003 — CRUD Fábrica ABM

**Estado**: Implementado
**Fecha**: 2026-03-05
**Solicitante**: Administración / Fábrica

---

## Descripción de la idea

El administrador necesita poder cargar y mantener actualizado el catálogo de fábrica (extrusoras, líneas, productos, marcos, hojas, interiores, perfiles, accesorios, vidrios, tratamientos) desde la interfaz web, sin depender de acceso directo a la base de datos.

## Por qué es valioso

Hoy los cambios en la configuración de fábrica requieren acceso técnico a la DB. Tener un ABM web reduce la dependencia del equipo técnico para tareas de configuración del negocio.
