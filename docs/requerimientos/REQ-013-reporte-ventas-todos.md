# REQ-013 — Reporte de ventas muestra Blanco y Negro

**Estado:** En desarrollo
**Fecha:** 2026-04-23

## User Story
Como usuario quiero que el reporte de ventas muestre tanto ventas "Blanco" como "Negro" para que los totales coincidan con el listado general de ventas.

## Criterios de Aceptación
- El reporte de ventas muestra ventas de ambos tipos ("Blanco" y "Negro").
- El total de ventas y la cantidad de ventas coinciden con el listado general filtrado por fecha de pago.
- El detalle de ventas en el reporte incluye una columna que indica el tipo ("Blanco"/"Negro").
- El usuario puede distinguir fácilmente entre ambos tipos en el reporte.

- Complejidad: Pequeño
- Derivará en una feature cuando se implemente.
